# TDE: Desenvolvimento de Games com p5.js - Galaga Clone

Este projeto consiste em um clone do jogo clássico **Galaga**, desenvolvido como parte do TDE para a disciplina de Web Development: HTML5 Canvas e Games. O jogo foi construído utilizando a biblioteca **p5.js** e foca na aplicação de conceitos de lógica de programação, manipulação de objetos, listas e gerenciamento de assets.

## Instruções de Uso

Para jogar, siga os passos abaixo:

1. **Acesso ao Jogo:**
   * Copie o codigo em `sketch.js` e execute o projeto através do [p5.js Web Editor].
   * Em sketch files, crie um folder com o nome "assets" e dentro de assets crie "image" e faca o upload das imagens dentro de image

2. **Comandos:**
   * **Tecla ESPAÇO:** Inicia o jogo a partir da tela de abertura ou reinicia após um Game Over.
   * **Setas Teclado (Esquerda / Direita):** Movimentam a nave do jogador horizontalmente pela tela.
   * **Tecla X:** Dispara projéteis para destruir as naves inimigas.
   * **Tecla ENTER:** Reinicia a partida quando a tela de "Nave Destruída" for exibida.

3. **Objetivo:**
   * Destrua o maior número possível de naves inimigas para acumular pontos.
   * Evite que os inimigos toquem sua nave ou alcancem a parte inferior da tela, caso contrário, o jogo terminará.
